Corporate Contributors License Agreement

If you are submitting code where the copyright is held by a corporation, then you need a signed CCLA. DWR joined the Dojo Foundation in 2007, and since then, they have been handling legal matters for us, including managing CLAs. As a result of this rather than signing the Getahead CCLA, we now request Dojo Foundation CCLAs. Since both CCLAs were based on the Apache CCLA, there is very little change.

  • Contributors License Agreement (PDF)
  • Corporate Contributors License Agreement (PDF)

The Dojo Foundation can accept fax, mail, or email a scans of the signed document.

Processing note: If you send a scanned CLA to the Dojo Foundation, then please copy [joe at getahead dot org]. That way we know to which foundation project the CLA is primarily aimed at.
