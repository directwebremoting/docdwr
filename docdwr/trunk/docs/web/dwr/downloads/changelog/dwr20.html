DWR version 2.0.7

Our latest 2.0.7 release is a minor bug fix release.

  • [DWR-60] - Synchronized block in DefaultScriptSessionManager locking entire appserver
  • [DWR-144] - DWR annotation with IBM JDK 1.4 makes dwr throws an exception when first ajax call is done
  • [DWR-291] - deadlock between DefaultScriptSession and JettyContinuationSleeper
  • [DWR-446] - A timeout with id 0 is not cleared
  • [DWR-461] - DefaultConverterManager.converters not synchronized
