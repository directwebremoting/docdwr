Development Log

The purpose of this page is to provide the community with the development team's status. If you find that this page has not been updated in awhile that does not mean development has stopped - we may just be very busy. There are several others ways to check for activity - JIRA, Bamboo, or our mailing list to see recent activity on the project.

October 7, 2010

We are very close to an RC2 release that we are planning for later this month. We are hoping to follow this release up with an official 3.0 release sometime early 2011. The official 3.0 release will be missing some of the reverse AJAX issues we have been working. They will be in version 3.0.1.

December 18th, 2009

Over the last few months we have been very busy on admistrative tasks. We have moved our hosted sites over to a Dojo Foundation VPS. This has allowed us a lot more control. We have configured bamboo on the VPS and we now have continuous integration builds up and running. We have also set-up a Maven repository and are currently working on getting our builds automatically synched with Maven central (thanks to Henric Larsson). We have completed several issues for a 2.0.6 release which will be out shortly. We have also made progress on deadlock and concurrency issues (thanks to Tim Peierls). I am currently working on integrating some of Tim's recommendations into our codebase.

July 9th, 2009

We are still slowly working toward an RC2 release. We currently have one unresolved issue scheduled for RC2 which is related to a deadlock in some of the sleeper code. We have started working on release notes and updated documentation for the RC2 release. Since the last update we have resolved approximately 20 issues.

May 20th, 2009

We are currently working on a plan to get the RC2 release out as soon as possible. We currently have 25 outstanding issues for RC2, but many of these issues either affect a very small percentage of the user base or are not critical.

Progress since April 20th, 2009

We have resolved approximately 20 issues since April 20th and we have made several improvements to the documentation (new changes have not been published).
