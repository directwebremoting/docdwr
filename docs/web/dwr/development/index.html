Development Resources

There are pages on this site, describing how to get involved and who the current development team is.

A number of sites host DWR resources:

  • The DWR source repository is hosted on GitHub and is available at https://github.com/directwebremoting/dwr
  • The mailing list archive. There are several mailing lists, which you can subscribe to.
  • Our bug tracking database: DWR JIRA.
  • The Dojo Foundation hosts our continuous integration build server: DWR Bamboo server.
