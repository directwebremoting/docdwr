Supported Environments

DWR requires JDK 1.3 or later and a servlet engine that supports servlet spec version 2.2 or later. Earlier versions of the servlet spec may also work.

It is frequently tested in many operating environments including:

  • Tomcat
  • Weblogic
  • Websphere
  • JBoss
  • Jetty
  • Resin
  • Sun ONE
  • Glassfish
