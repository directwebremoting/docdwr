Support

If you are having trouble getting DWR working we ask that you please follow the steps below to find a resolution - Thank you!

If you fail to follow these steps it is likely that any inquiries will result in a link back to this page!

  1. Step 1: Site resources - This site contains information to solve many if not most problems. Please look for an answer here first! You can use our site search or look at the following pages which we highly recommend:
    1. On the Solving Common Problems page we outline several of the more common issues encountered while using DWR. We also provide information on how to troubleshoot and solve these problems. If you read this page and follow the advice given it is likely you will be able to solve the problem on your own.
    2. The Getting Started guide explains how to configure and start to use DWR.
    3. For answers to some of the most common questions we receive see the Answers to Common Questions page.
  2. Step 2: Mailing list search - Chances are you are not the first person to encounter your problem. Search the mailing list to see if someone has had a similar problem. You can find mailing list information here.
  3. Step 3: Ask the mailing list - If a search of our mailing lists did not help, submit a question to the list. Our lists are very active and you are likely to receive an answer in a short period of time. You can find mailing list information here. Please make sure you include the version of DWR you are using as well as any information you have gathered from our troubleshooting page.
  4. Step 4: File a bug report - If the previous three steps have not helped we host our own bug tracking database: JIRA on DWR. This system is for bugs only and if you have failed to follow the above steps it is likely that your issue will be closed immediately and you will be directed to this page!

Mailing lists

The DWR mailing lists are hosted by java.net. You can join Java.net by registering here, and then you can subscribe. However, you should not need to subscribe to take part - web interfaces are available. Please see this page if you need help subscribing to or sending an email to one of our lists.

DWR-Users Mailing List

The users mailing list is where the vast majority of discussion about DWR take place. If you have a general question or are encountering a problem this is the mailing list you should use.

Before sending a message to the list please make sure you have followed the instructions at the top of this page. Also make sure you include the version of DWR you are using as well as any information you have gathered while troubleshooting your problem.

You can send email to the list directly using: [users at dwr.dev.java.net], or you can access the list on the web:

  • Java.net archive of DWR-Users
  • Nabble archive of DWR-Users
  • GMane archive of DWR-Users

DWR-Development

This list is used for DWR developer discussions. This list should not be used for questions (please use the users list for these types of inquiries!). The email address for the list is: [dev at dwr.dev.java.net], or, on the web:
  • Java.net archive of DWR-Dev
  • Nabble archive of DWR-Dev
