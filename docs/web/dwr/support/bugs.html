We host our own bug tracking database: JIRA on DWR. Please note this is for actual bugs - not questions, comments, etc. The process for creating a bug in JIRA should be:

  1. Search for your issue on the user's mailing list
  2. If you cannot find your issue submit a new topic to the list.
  3. If you have found a legitimate issue we will direct you to create an issue in JIRA
